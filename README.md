6. 學生學習成果認證與展示系統 
核心功能： 
    學生功能 (成員 A/B)： 創建個人資料 (照片、簡介)。 
    創建學習成果 (CRUD)：學生可以新增自己的擅長科目、程式語言、參與競賽、取得證照。 
    (重點) 所有新增的成果，初始狀態皆為「待審核」。 學生只能編輯/刪除自己「待審核」的項目。 

    管理員/老師功能 (成員 C)： 擁有管理員帳號，登入後台。 查看所有學生的「待審核」成果列表。 
    (重點) 可以將成果的狀態修改為「已認證」或「不通過」。 訪客功能 (所有人)： 公開的「人才搜尋」頁面。 
    
    訪客可以搜尋（例如：搜尋具備 "PHP" 和 "MySQL" 技能的學生），但搜尋結果只會顯示「已認證」的成果。
